# Hydra


## Installation
- Get the a list of virtual environments: `conda env list`
- Chose one virutal env: `conda activate <my_virtual_env>`
- Install hydra: `pip instll hydra-core`
- Install the hydra colour plugin: `pip install hydra_colorlog`
